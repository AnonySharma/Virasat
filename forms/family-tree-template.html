Family Tree — Form template

Use this set of questions to build a Google Form that your relatives can fill out. Their responses go into a Google Sheet — download it as CSV and import here.

How to set this up in 5 minutes

  1. Open forms.new in your Google account.
  2. Title your form (e.g. "Sharma Family Tree — share your details").
  3. Add each question below. Set the type as indicated. Mark required ones as required.
  4. In the form's Responses tab, click Link to Sheets to create a spreadsheet.
  5. Share the form link with your family. As responses arrive, open the linked sheet and choose File → Download → Comma-separated values (.csv).
  6. Back in the family tree app, click Collect → Import CSV responses to load them.

Tip: in Google Forms, use the Section feature to group personal info, parents, and stories. For long-text answers (description, achievements, education) choose Paragraph.

Questions to add

1. Full name (English) required Short answer
Example: Ramesh Sharma. This must match the parent's/spouse's name fields exactly so we can link relations.
2. नाम (हिन्दी) Short answer · optional
Same name written in Devanagari, if you'd like the Hindi UI to use it.
3. Father's full name Short answer
Use exactly the spelling as recorded in the family tree. Leave blank if unknown.
4. Mother's full name Short answer
5. Spouse's full name Short answer
Leave blank if not applicable.
6. Birth date Short answer
YYYY-MM-DD if known. Just the year (YYYY) is also fine.
7. Death date Short answer · optional
Leave blank if living.
8. Birth place Short answer
City, country.
9. Death place Short answer · optional
10. Gender Multiple choice
Options: Male / Female / Other / Prefer not to say.
11. Occupation Short answer
12. About / short description Paragraph
A paragraph or two — personality, hobbies, memories.
13. Life achievements Paragraph
One per line, or separated by semicolons. e.g. "Founded the village school; First in family to graduate".
14. Education Paragraph
One per line, or separated by semicolons. e.g. "MA Hindi, University of Rajasthan, 1958".

Apps Script — auto-create form (optional)

Want to skip the manual setup? Open script.google.com, create a new project, paste this script, and run buildForm(). It generates the form and links a sheet for you. 

function buildForm() {
  const form = FormApp.create("Family Tree — Share your details");
  form.setDescription("Help us build the family tree. Your responses go into our shared Sheet.");

  const qs = [
    ["Full name (English)", true],
    ["नाम (हिन्दी)", false],
    ["Father's full name", false],
    ["Mother's full name", false],
    ["Spouse's full name", false],
    ["Birth date (YYYY-MM-DD or YYYY)", false],
    ["Death date (leave blank if living)", false],
    ["Birth place", false],
    ["Death place", false],
  ];
  qs.forEach(([title, req]) => form.addTextItem().setTitle(title).setRequired(req));

  form.addMultipleChoiceItem()
      .setTitle("Gender")
      .setChoiceValues(["Male", "Female", "Other", "Prefer not to say"]);

  form.addTextItem().setTitle("Occupation");
  form.addParagraphTextItem().setTitle("About / short description");
  form.addParagraphTextItem().setTitle("Life achievements (one per line)");
  form.addParagraphTextItem().setTitle("Education (one per line)");

  // Link a fresh sheet
  const ss = SpreadsheetApp.create("Family Tree — Responses");
  form.setDestination(FormApp.DestinationType.SPREADSHEET, ss.getId());

  Logger.log("Form: " + form.getPublishedUrl());
  Logger.log("Sheet: " + ss.getUrl());
}

Tip: keep the question titles unchanged — the importer matches CSV headers to these names.
